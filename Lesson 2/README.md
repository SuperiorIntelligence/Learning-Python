# 🐍 Python Programming — Session 02: Getting Started & Python Fundamentals

A beginner-friendly Jupyter Notebook introducing core Python concepts, from environment setup to operators. Part of an ongoing Python programming course.

---

## 📋 Topics Covered

| # | Topic | Summary |
|---|-------|---------|
| 1 | 🛠️ VSCode + Extensions | Installing VSCode, Python, and key extensions (Python, Jupyter, Pylance) |
| 2 | 📄 `print()` | Displaying output with separators and line-ending control |
| 3 | 💬 `input()` & f-strings | Reading user input, type conversion, and string formatting |
| 4 | 🏗️ Syntax & Indentation | Python's 4-space block structure and case sensitivity |
| 5 | 💡 Comments & Docstrings | Inline `#` comments and `"""..."""` docstrings |
| 6 | 📦 Variables & Types | `str`, `int`, `float`, `bool`; `type()`, `len()`, type conversion |
| 7 | ⚖️ Operators | Comparison (`==`, `!=`, `>`, `<`, `>=`, `<=`) and logical (`and`, `or`, `not`) |

---

## 🎯 In-Class Exercises

Five hands-on exercises are included at the end of the notebook:

1. **Personal Profile Card** — collect user info and display a formatted card using f-strings and `len()`
2. **Type Detective** — predict the output of type conversion expressions before running them
3. **BMI Calculator** — compute BMI from user input using arithmetic and comparison operators
4. **Logic Gates Simulator** — simulate AND, OR, and NOT gates using boolean inputs
5. **Password Validator** — check password length rules using comparison and logical operators

---

## ✅ Learning Objectives

By the end of this session, learners will be able to:

- Set up a Python development environment with VSCode
- Use `print()`, `input()`, and f-strings to build interactive programs
- Understand Python's indentation rules and block structure
- Declare variables, identify data types, and convert between them
- Apply comparison and logical operators to evaluate conditions

---

## 🔧 Requirements

- Python 3.6+
- VSCode with the **Python** and **Jupyter** extensions
- No prior programming experience required
